# PDF to TXT Converter

This repository provides simple Python scripts to automate the conversion of PDF files into plain text (`.txt`) format.  
It is designed for users who want to quickly extract text from PDF documents and save them in a reproducible, editable format.

---
# To convert a SINGLE file, PDF to TXT
USE THIS SCRIPT
pdftotxt.py
---
---
# To convert all PDF files in the folder, PDF to TXT
USE THIS SCRIPT
pdftotxt-entirefolder.py
---

## ✨ Features
- Sorts files in a target folder by name.
- Automatically selects the first PDF file in the sorted list.
- Extracts text from **all pages** (page one to last).
- Creates a `.txt` file with the same name as the PDF.
- Saves the extracted content into the new text file.
- Closes the PDF file after processing.
- Prints the **number of files** in the folder for quick reference.
